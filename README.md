# DevConnect Pro

AI-powered Developer Marketplace

## Features
- User Registration (Developer/Company)
- Project Posting
- Project Marketplace
- FastAPI Backend
- Streamlit Frontend
- JWT-ready architecture
- Docker Support

## Tech Stack
FastAPI | Streamlit | SQLite | SQLAlchemy | Docker

## Run Backend
cd backend
uvicorn main:app --reload

## Run Frontend
cd frontend
streamlit run app.py
